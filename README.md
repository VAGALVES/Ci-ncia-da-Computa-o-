# CC Study OS · VS Consulting

Material interativo de consulta, estudo e portfólio para Ciência da Computação.

## O que é

Um app estático em HTML, CSS e JavaScript puro, com:

- disciplinas de Ciência da Computação;
- busca e filtros por trilha;
- modo foco;
- tema claro/escuro;
- cards de estudo;
- exemplos demonstrados;
- entregáveis técnicos profissionais;
- navegação mobile;
- estrutura pronta para Netlify.

## Deploy no Netlify

Como é um projeto estático:

- Build command: deixe vazio
- Publish directory: `.`
- Arquivo principal: `index.html`

## Estrutura

```txt
.
├── index.html

```

## VS Consulting

Este projeto faz parte da linha de materiais vivos de estudo, portfólio e formação profissional.


## Versão 0.6 · CHA

Esta versão adiciona uma camada pedagógica baseada em **CHA — Conhecimento, Habilidade e Atitude**.

Cada disciplina passa a conter:

- Matriz CHA da disciplina;
- práticas e desafios progressivos;
- níveis de domínio;
- estudo guiado;
- temas essenciais demonstrados;
- mapa de dependências;
- entregáveis técnicos profissionais.

O objetivo é transformar cada disciplina em uma unidade de formação completa: conceito, prática, postura e portfólio.
